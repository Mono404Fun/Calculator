# ExprEval

A high-performance expression evaluator and calculator library written in C++.

## Features

- **Expression Parsing & Evaluation**: Parse and evaluate mathematical expressions with support for standard operators and functions
- **Efficient Computation**: Optimized C++ implementation for fast expression evaluation
- **Cross-platform**: Builds and runs on Linux and Windows

## Quick Start

### Prerequisites

- C++ compiler (C++11 or later)
- CMake or build scripts provided

### Build

Navigate to the `calculator/` directory and run:

**Linux:**
```bash
./build.sh
```

**Windows:**
```batch
./build.bat
```
